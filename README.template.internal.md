Para rodar este projeto localmente, siga os passos abaixo.

### Pré-requisitos
- Node.js (v18+)
- npm ou yarn

### Instalação
1.  **Clone o repositório:**
    ```bash
    git clone {{URL_REPOSITORIO}}.git
    ```
2.  **Instale as dependências:**
    ```bash
    cd {{SLUG_PROJETO}} && npm install
    ```
